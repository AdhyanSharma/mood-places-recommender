# 🌍 Mood Places Recommender

AI-powered travel recommendation web application that suggests nearby places based on user mood using NLP sentiment analysis, voice recognition, and geolocation.

## 🚀 Features

- 🎤 Voice mood detection
- 😊 Mood-based recommendations
- 📍 Location-aware nearby places
- 🧠 NLP sentiment analysis (TextBlob)
- 🗺 Google Maps integration
- ⚡ FastAPI backend
- 🎨 React + Tailwind frontend

## 🧱 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- FastAPI
- Python
- TextBlob NLP
